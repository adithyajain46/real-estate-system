=======================================================
HOW TO EXPLAIN THIS PROJECT TO YOUR COORDINATOR
=======================================================

"This is a web-based CRM system for real estate businesses built using
Laravel as the backend framework, Bootstrap 5 for the frontend,
and MySQL as the database. The system has 5 main modules:

1. Authentication — users can register and login securely
2. Dashboard — shows total properties, clients, available listings,
   and portfolio value with doughnut charts
3. Properties — full CRUD with image upload, search by title/location,
   and filter by status and type
4. Clients — manage buyers, sellers and tenants with contact details
5. Reports — bar charts and pie charts showing property distribution
   by type and status using Chart.js

The backend uses Laravel's MVC architecture with Eloquent ORM for
database operations, resource controllers for clean routing, and
Blade templating engine for views."
